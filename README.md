# SIMPLE PAINT :art:

Simple Paint predstavlja program za crtanje i jednostavnu manipulaciju slikama, po ugledu na postojeći program Paint.

## Uslovi: :pencil:
* ```g++ [c++17]```
* ```Qt 5.10```
